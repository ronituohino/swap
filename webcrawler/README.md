# Web Crawler

## Prerequisites

- Python (>v3.10)
- [uv](https://docs.astral.sh/uv/) (>0.5.21)

## Installation

- `uv sync`

## Run

- `uv run scrapy crawl infinite`

#### Additional arguments

- `-s CLOSESPIDER_PAGECOUNT=<pages>`
- `-s CLOSESPIDER_ITEMCOUNT=<items>`
- `-s CLOSESPIDER_TIMEOUT=<seconds>`
