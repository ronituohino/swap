# Web crawler

## Prerequisites

- Python (>v3.10)
- [uv](https://docs.astral.sh/uv/) (>0.5.21)

## Installation

- `uv sync`

## Run

- `uv run scrapy crawl infinite`

#### Additional arguments

- `-a start_urls="<site1>|<site2>|<etc.>"`
- `-a categories="<category>:<keyword1>,<keyword2>|category1:<keyword1>,<keyword2>|<etc.>"`
